# Мафия Онлайн

Этот репозиторий содержит код веб-приложения для игры в "Мафию" онлайн.


## Описание

Данное приложение позволяет играть в "Мафию" через веб-браузер. Игроки могут присоединяться к игровым сессиям, получать роли, голосовать и взаимодействовать друг с другом, чтобы определить, кто из них Мафия.
